## 2. 로그인 페이지 개선

### 2.1 현재 디자인 문제점

- 소셜 로그인 버튼이 작고 구분이 어려움
- 에러 메시지 위치가 일관되지 않음

- 중요한 액션이 숨겨져 있음

## 4. 피드백 페이지 개선

- 영상과 댓글 영역 분리가 불명확
- 타임스탬프 기반 피드백 불가
- 피드백 상태 추적 어려움
