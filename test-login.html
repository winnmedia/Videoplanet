🧪 로그인 버튼 Hydration 테스트

📋 테스트 방법:

  1. 페이지 로딩: 아래 iframe에서 로그인 페이지가 로드됩니다
  2. 초기 상태: 버튼이 "준비 중..." 으로 표시되어야 합니다
  3. Hydration 완료: 1-2초 후 "로그인" 버튼으로 변경되어야 합니다
  4. 클릭 테스트: 버튼 클릭 시 콘솔에 로그가 출력되어야 합니다
🔄 테스트 준비 중...

🎯 예상 결과:

  • ✅ SSR: "준비 중..." 버튼 (disabled, opacity: 0.7)
  • ✅ Hydration 완료: "로그인" 버튼 (enabled, opacity: 1)
  • ✅ 클릭 시: 콘솔에 '[Login] Button clicked - isHydrated: true' 출력
  • ✅ API 호출: authApi.signIn 함수 실행
